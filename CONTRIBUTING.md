# Contributing

## Requirements

- Node.js 20+

## Setup

```bash
npm install
```

## Common scripts

```bash
npm run dev
npm run lint
npm run typecheck
npm run test
npm run test:e2e
npm run test:a11y
npm run format
```

## Percy

Set `PERCY_TOKEN` in your environment and run:

```bash
npm run test:percy
```
